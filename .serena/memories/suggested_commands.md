# Suggested Commands

## Development
- **Install Dependencies:** `bun install`
- **Build:** `bun run build` (Builds both CJS and ESM formats)
- **Lint:** `bun run lint`
- **Test:** `bun test`

## Publishing
- **Publish:** `bun run publish` (Builds and publishes to npm)

## System Utilities (macOS/zsh)
- `ls`, `cd`, `grep`, `find`
- `git status`, `git add`, `git commit`, `git push`
