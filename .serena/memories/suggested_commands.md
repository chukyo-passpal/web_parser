# Suggested Commands
- `bun install` — install dependencies (Bun-managed).
- `bun test` — execute Bun test suite (`bun:test`).
- `bun run scripts/generate-expected.ts` — refresh JSON fixtures after parser updates.
- `pwsh` basics: `Get-ChildItem` (alias `ls`) to inspect folders, `Set-Location <path>` (alias `cd`) to move around, `git status` to review pending changes.
